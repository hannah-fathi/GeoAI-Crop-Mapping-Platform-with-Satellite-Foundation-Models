# Experimental Design


## 1. Objective


The experiments evaluate the capability of satellite
representations for agricultural crop classification.


---

# 2. Experimental Tasks


Two experiments were designed:


## Experiment 1

Vegetation Detection


Objective:

Separate vegetation and non-vegetation polygons.



## Experiment 2

Crop Classification


Objective:

Classify agricultural fields into five crop categories.



---

# 3. Feature Configurations


Evaluated feature sources:


- AlphaEarth 64-dimensional embedding

- AlphaEarth statistical embedding

- Galileo representation

- AlphaEarth + Galileo fusion



---

# 4. Model Configuration


Machine Learning Model:

Multi-Layer Perceptron


Training:

Supervised learning


Dataset Split:

70% Training

30% Testing


Sampling Strategy:

Stratified split


---

# 5. Evaluation Outputs


Generated outputs:


- Accuracy
- Precision
- Recall
- F1-score
- Confusion analysis
- GIS prediction maps
