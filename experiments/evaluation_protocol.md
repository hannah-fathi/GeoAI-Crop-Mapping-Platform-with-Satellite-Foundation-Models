# Evaluation Protocol


## 1. Evaluation Strategy


The evaluation follows a supervised classification
protocol.


Data is divided into:


Training Set:

70%


Testing Set:

30%



---

# 2. Classification Metrics


The following metrics are reported:


## Accuracy


Measures overall prediction correctness.



## Precision


Measures reliability of positive predictions.



## Recall


Measures detection capability.



## F1-score


Provides balance between precision and recall.



## Macro F1


Used to evaluate performance across all classes
without class imbalance bias.



---

# 3. Experimental Results


The crop classification model achieved:


Accuracy:

94%


Macro F1-score:

94.05%



These results demonstrate the effectiveness of
satellite embedding based crop classification.


---

# 4. Reproducibility Note


Due to private dataset restrictions,
raw agricultural data cannot be distributed.


However, the complete computational workflow,
methodology, and evaluation framework are provided.
